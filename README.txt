## Installation
To run this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Ziglarski/pagesrepo
   ```
2. Open the `index.html` file in your preferred web browser.

## Usage
This website provides:
- Information on services offered.
- A contact form for inquiries.
- Links to social media profiles.
- Blog updates and industry insights.

## SEO Optimizations
This website includes various SEO optimizations:
- Proper use of meta tags (description, keywords, author).
- Canonical links to prevent duplicate content issues.
- Structured data for enhanced search engine visibility.
